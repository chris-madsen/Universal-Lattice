# 01_raw_24cell_web

## Hypothesis
Current lattice is a subset of a raw 2D web from some projection of the 24-cell.

## Why It Is Plausible
The 24-cell already supplies a dense family of projected edges and may directly produce the observed web structure.

## Witness
A concrete projection whose raw projected edge-web contains the current lattice as a strict or structural subset.

## Kill Criteria
Node signatures or direction-family counts are incompatible before any meaningful fit is possible.

## Stages
- baseline
- prune
- search
- analyze
- finalize
