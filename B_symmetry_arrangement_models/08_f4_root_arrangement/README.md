# 08_f4_root_arrangement

## Hypothesis
Current lattice is better explained by an F4 root or hyperplane arrangement than by a single polytope shadow.

## Why It Is Plausible
The lattice behaves like an arrangement of line families rather than just a set of projected edges.

## Witness
An F4-derived arrangement model that matches the central grammar and local fan structure naturally.

## Kill Criteria
Arrangement-level geometry still fails to reproduce the required node and fan combinatorics.

- baseline
- prune
- search
