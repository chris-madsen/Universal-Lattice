# 09_f4_b4_hybrid

## Hypothesis
Current lattice is a hybrid object where different substructures are better explained by F4 and B4/C4 respectively.

## Why It Is Plausible
The vertical scaffold can look cubical while diagonal families can look 24-cell-like.

## Witness
A mixed-symmetry explanation with a cleaner structural account than either pure model alone.

## Kill Criteria
The hybrid adds complexity without improving the structural account.

- baseline
- prune
- search
- analyze
- finalize
